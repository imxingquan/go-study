GO 安装
1. 解压安装文件
	tar -C /urs/local -xzf go1.6.linux-amd64.tar.gz
2. 设置环境变量 $HOME/.profile
	export PATH=$PATH:/usr/local/go/bin
安装到自定义位置
	tar -C ~/go -xzf go1.6*.gar.gz
	设置GOROOT变量到安装的自定义目录
	export GOROOT=&HOME/go
	export PATH=$PATH:&GOROOT/bin


参考：https://golang.org/doc/
